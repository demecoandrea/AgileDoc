# AgileDoc 📄🚀

**AgileDoc** è uno strumento intuitivo e potente progettato per la creazione e la gestione rapida di file PDF da molteplici sorgenti. Che si tratti di digitalizzare documenti cartacei, importare file esistenti o creare pagine da zero, AgileDoc ottimizza il workflow grazie a un'interfaccia moderna e funzioni basate su Intelligenza Artificiale.

## ✨ Caratteristiche Principali

### 📥 Importazione Multisorgente
* **Drag & Drop Intuitivo:** Importa immagini e PDF trascinandoli direttamente dal computer.
* **Folder View Dinamiche:** Monitora cartelle locali in tempo reale. Puoi aggiungere o rimuovere viste, ordinando i file per data di modifica per trovare istantaneamente gli ultimi documenti prodotti.
* **Supporto Scanner:** Integrazione completa con scanner tramite protocolli **TWAIN** e **WIA**.
* **Appunti di Sistema:** Incolla immagini direttamente dalla clipboard.
* **Companion App Android:** Sincronizzazione con l'app dedicata per scattare foto ai documenti, con supporto a **MLKit** per un controllo avanzato dello scatto.

### 🛠 Editing e Ottimizzazione AI
* **Rilevazione Bordi Smart:** Integrazione con un modello **AI YOLO** per il rilevamento automatico dei bordi del documento.
* **Correzione Avanzata:** Strumenti per la correzione trapezoidale, regolazione di colore, luminosità e contrasto.
* **Editing di Pagina:** Crea contenuti misti aggiungendo caselle di testo, disegnando con il pennarello o evidenziando parti importanti.
* **Firme Digitali:** Apponi immagini con trasparenza (firme o timbri) direttamente sulle pagine prima della generazione.

### 📤 Output Personalizzato
* **Controllo Qualità:** Regola la risoluzione del file finale in DPI.
* **Gestione Media:** Scegli tra integrazione nativa dei file o rasterizzazione completa per massimizzare la compatibilità e ottimizzare il peso del PDF.

## 🛠 Tecnologie Utilizzate
* **Linguaggio:** Python
* **AI/ML:** YOLO (Object Detection), MLKit (lato Android)
* **Interfaccia:** PyQt
* **Protocolli:** TWAIN, WIA
